JAVA 8 新特性

JAVA 8 常见新特性

  • Lambda(包括更好的类型推断）

  • Stream API

  • 函数式接口

  • 方法引用与构造器引用

  • New Date/Time API(JSR 310)

  • 重复注解和类型注解

  • Optional 最大化减少空指针异常

  • 接口的静态方法和默认方法

  • Java编译器的新特性 – 参数名称(-parameters)

  • Hashorn JavaScript引擎 (支持jss）和类依赖分析器(jdeps)

  • 对Base64编码的支持

  • 并行数组

  • 并发性

  • JVM 新特性 – 使用Metaspace(JEP 122) 代替持久代(PermGen space)
